## 概要
> Efficient, expressive, elegant
>
> 公式サイト (https://nim-lang.org/) より
- プログラミング言語の一つ
- 静的型付けコンパイル型言語
- Pythonライクな文法
```nim
let message = "Hello, world!"
echo message   # Hello, world!
echo(message)  # Hello, world!

proc add(x, y: int): int =
  result = x + y

echo add(1, 2) # 3
```
